# Picture #
## 需求 ##
逛博客时发现了[这种效果](http://ghmagical.com/)(左侧鼠标悬停，切换图片背景)，于是想自己写一套类似效果的轮播图。
* 图片轮播
* 翻页效果
## 简介 ##
该博主的效果是用多张小图实现的，为了达到用组件来调用的效果，这里我就不区分小图了，底层用`background-image`实现切图。
## 实现功能 ##
* 鼠标悬停切换图片
* 图片切换时小图依次翻页(动效)
