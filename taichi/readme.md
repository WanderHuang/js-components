# taichi #
## 简介 ##
  纯css实现的太极图及动效
### 使用 ###

* 引入样式 `<link rel="stylesheet" href="./taichi.css">`

* 引入js  `<script src="./taichi.js"></script>`

* 挂载select完成初始化  

   ```js
   var tc = new Taichi({
       el: '#demo',
       size: 400,
       round: 'left'
   })
   
   ```
